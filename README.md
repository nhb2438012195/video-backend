项目名称：仿 B 站视频分享平台

项目描述：

基于 SpringBoot + Vue3 开发的视频分享平台，负责后端核心功能开发，实现视频上传、转码、存储、互动、搜索、缓存等企业级能力，专注于文件处理、异步任务、缓存优化、数据库设计、高并发接口等后端核心能力，采用前后端分离架构，Docker 容器化部署。

核心职责：

设计并实现用户、视频、评论、点赞、关注等模块的后端接口

使用 MinIO 对象存储管理视频文件，实现签名 URL 与防盗链

采用 Redis 缓存热点数据，优化接口性能，解决高并发问题

实现 Elasticsearch 全文检索，支持视频标题、简介搜索

完成 Docker 容器化部署与 Nginx 反向代理

实现大文件分片上传、断点续传、MD5 秒传，使用 MinIO 分布式存储

基于异步线程池处理视频转码等耗时任务，实现系统解耦

完成 MySQL 表结构设计、事务控制与索引优化

统一封装返回结果与全局异常处理，提升系统健壮性

技术栈：

SpringBoot、MySQL、Redis、MyBatis-Plus、MinIO、Elasticsearch、Docker、Nginx、JWT

1. 视频大文件上传
分片上传（前端切片 + 后端合并）
断点续传（断网重试）
秒传（文件哈希校验）
上传进度条
写简历关键词：
大文件分片上传、断点续传、文件 MD5 校验、秒传功能、文件合并策略
2. 视频转码系统
用 FFmpeg 做：
自动转码为 MP4
生成多种清晰度（360p/720p）
生成封面
异步转码（不阻塞页面）
写简历关键词：
FFmpeg 视频处理、异步任务、多清晰度转码、消息队列解耦
3. 存储服务
MinIO / 阿里云 OSS 对象存储
视频防盗链
过期签名 URL
写简历关键词：
对象存储、MinIO、OSS、URL 签名、防盗链
4. 高并发优化
Redis 缓存视频列表、用户信息
点赞、播放量统计
防止缓存穿透 / 击穿
写简历关键词：
Redis 缓存、缓存策略、热点数据优化、高并发接口、播放量计数
5. 互动功能（点赞、收藏、关注、评论）
点赞 / 取消点赞（幂等性）
关注 / 取关
评论嵌套
未读消息计数
写简历关键词：
关系设计、高并发写入、Redis 计数优化、消息通知
6. 搜索功能
Elasticsearch 视频搜索
标题 + 简介分词检索
高亮展示
