项目描述：

基于 SpringBoot + Vue3 前后端分离架构开发的视频分享平台，专注后端核心能力，实现视频存储、异步处理、高并发接口、缓存优化、数据库设计等企业级能力，项目采用 Docker Compose 容器化部署。

涉及技术：Java、SpringBoot、MyBatis-Plus、MySQL、Redis、Redisson、MinIO、Elasticsearch、Docker、Docker Compose、Nginx、JWT、SpringDoc

一、职责描述
1. 负责用户、视频、评论、消息、后台管理等核心业务模块的接口设计与开发工作，保障模块功能正常落地。
2. 负责视频文件存储、上传相关功能的落地，优化上传体验与稳定性，保障文件存储安全。
3. 负责系统性能优化，解决并发安全问题，提升接口响应速度与数据查询效率。
4. 负责数据库、检索引擎相关设计与优化，保障数据存储安全、查询高效。
5. 负责系统健壮性与可维护性提升，规范接口文档，完成项目容器化部署与运维相关工作。

二、技术亮点
1. 使用 Redisson 分布式锁解决重复上传、重复点赞等并发安全问题，保障数据一致性。
2. 基于 MinIO 实现视频文件存储，完成签名 URL 生成与防盗链策略；实现大文件分片上传、断点续传、MD5 秒传功能，提升上传稳定性与效率。
3. 使用自定义线程池异步处理视频转码任务，实现耗时业务与核心业务解耦。
4. 基于 Elasticsearch 实现视频标题、简介的全文检索功能，提升检索效率与准确性。
5. 采用 Redis 缓存热点数据，优化接口性能，降低数据库压力；完成 MySQL 表结构设计、事务控制与索引优化，提升查询性能。
6. 基于 Redisson + AOP 实现分布式接口限流，防止流量突刺与恶意请求，提升服务容错能力。
7. 引入 RabbitMQ 消息队列 异步解耦点赞、评论、通知等业务，实现削峰填谷与可靠消息处理。
8. 使用 AOP 切面 实现全局操作日志记录与细量级 RBAC 权限控制，完善系统安全与审计能力。
9. 集成分布式定时任务，实现缓存刷新、数据统计、历史数据清理等自动化运维能力。
10. 集成 SpringDoc 自动生成 OpenAPI 标准接口文档，实现多核心业务模块接口的设计与开发；封装统一返回体与全局异常处理，提升系统健壮性与可维护性。
11. 使用 Docker + Docker Compose + Nginx 完成项目容器化部署，保障服务稳定运行。


视频投稿：

1. 前端先分片上传视频到 MinIO
不上传到你的 Java 服务器！
直接传 MinIO / 后端签名直传
后端只生成一个 uploadId 或唯一 key
2. 上传过程中 / 上传完，用户填写标题、封面、分类
完全异步，不阻塞
上传和填表并行
3. 用户点【发布】
前端把：
key（后端给的，不是自己编的）
标题、简介、封面、分类
一起提交给后端
4. 后端校验
key 是否存在
是否属于当前用户
是否有效（未被清理）
5. 校验通过 → 写入 MySQL → 投稿成功
6. 后端异步清理垃圾文件
超过 24 小时未发布的临时文件
定时任务删除
