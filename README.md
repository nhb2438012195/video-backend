项目名称：基于 SpringBoot + Vue3 的视频分享平台

项目描述：

独立开发的一个轻量级视频分享平台，实现了视频上传、转码、播放、评论及用户社交等核心功能。通过该项目深入实践了前后端分离架构、分布式系统设计思想，解决了文件上传优化、高并发缓存、异步处理等技术挑战，并采用Docker完成容器化部署。

核心技术：java，SpringBoot , MyBatis-Plus, MySQL , Redis , MinIO, RabbitMQ 

核心职责与技术实现：

1.对热门视频数据进行缓存，采用Cache-Aside模式，先读缓存，缓存未命中时读数据库并回填，为防止缓存击穿，对热点视频设置逻辑过期时间。

2.引入RabbitMQ实现任务队列，视频上传完成后，发送消息到转码队列，消费者异步执行FFmpeg转码，生成多分辨率版本，转码完成后发送通知，更新视频状态

3.使用 Spring Security + JWT 实现认证鉴权，完成用户身份认证、接口权限控制，支持无状态分布式访问，保障系统接口安全。

4.基于MinIO实现分片上传架构，前端将文件按10MB分片，逐片上传至MinIO，服务端记录上传进度，支持断点续传，通过文件MD5校验确保分片完整性，采用预签名URL机制，前端直传MinIO，减轻后端带宽压力

5.设计并实现热点榜、点赞榜动态排名体系，采用 Redis ZSet 存储热度分值，完成动态权重计算、热度衰减、定时刷新，支撑高并发下榜单快速查询与实时排名展示。

6.基于 Redis Set 实现关注 / 取关与共同关注功能，利用集合交运算高效查询，减少 DB 访问，提升性能。

7.实现基于 WebSocket 的用户实时一对一聊天功能，支持消息即时推送、未读消息计数、在线状态感知，保证消息实时性与可靠性。
